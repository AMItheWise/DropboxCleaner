# Changelog

All notable changes to this project will be documented in this file.

## 1.0.0

- Initial public release
- Tkinter desktop app for connection, job setup, progress, and results
- Shared Python backend used by both GUI and CLI
- Dropbox inventory, cutoff filtering, dry-run planning, copy staging, and verification
- SQLite-backed resumability and structured logging
- CSV, JSON, and Markdown outputs for audit and reporting
- Automated test coverage for pagination, filtering, conflicts, retries, verification, and resume paths
