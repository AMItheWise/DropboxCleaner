# Code of Conduct

## Our Standard

We want Dropbox Cleaner to be a respectful, practical, and inclusive open-source project.

Examples of behavior that contributes to a positive environment:

- Being respectful in technical discussion
- Giving actionable, specific feedback
- Assuming good intent while still holding a high engineering bar
- Focusing on user safety, privacy, and reliability

Examples of unacceptable behavior:

- Harassment, abuse, or personal attacks
- Trolling, intimidation, or bad-faith engagement
- Publishing private information without consent
- Repeatedly derailing discussion away from the project

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors whose behavior violates this code of conduct.

## Scope

This code of conduct applies in:

- GitHub issues
- Pull requests
- project discussions
- community spaces directly tied to the project

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainers through the repository's security or contact channel.
